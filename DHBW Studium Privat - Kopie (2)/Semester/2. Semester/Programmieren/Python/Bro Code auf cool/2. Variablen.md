- Ein Container für eine Value. Verhält sich wie der Wert dem ihm zugesprochen wird.

### Strings
```python
name = "Bro"
print("Hello" + name)
print(type(name))

#Output:
"Hello Bro"
"<class 'str'>"

first_name = "Bro"
last_name = "Code"
ful_name = first_name + " " + last_name

print("Hello" + full_name)

#Output:
"Hello Bro Code"
```

### Integer
```python
age = 21
age += 1
print("Your age is: " + str(age))
print(type(age))

#Output:
"Your age is: 22"
"<class 'int'>"
```

## Float
```python
height = 250.5
print("Your height is: " str(height) + "cm")
print(type(height)

#Output: 
"Your height is: 250.5 cm
"<class 'float'>"
```
